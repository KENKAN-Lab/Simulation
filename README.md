# 建築環境物理計算

## 水蒸気

### 飽和水蒸気圧の計算
温度から飽和水蒸気圧および飽和水蒸気圧を温度で微分した値を計算する。
飽和水蒸気圧の計算式は以下に対応している。
* SONNTAG(ZIS Z8806,1995)
* WMO (Goff, 1957)
* Wexler-Hyland (1983)
* Tetens(1930)
* Briggs and Sacket(1989)
* Antoine
* Goff Gratch
